### 工程介绍
- spring-cloud-alibaba-version-parent 统一依赖管理工程
- spring-cloud-alibaba-common 公共服务依赖
- spring-cloud-alibaba-service-user 用户中心业务服务
- spring-cloud-alibaba-service-member 会员中心业务服务
